# SlimeTools

SlimeTools is a multipurpose Minecraft datapack designed to simplify server
management while offering quality-of-life utilities for players and admins.

## Features

- **Administrative helpers** – Quickly configure gamerules, teleport players, and
  manage world state without diving into raw commands.
- **Utility tools** – Access frequently used functions such as inventory cleanup,
  location bookmarking, and resource tracking.
- **Quality-of-life tweaks** – Offer optional conveniences like custom messages,
  chunk loaders, or anti-grief utilities depending on the server's needs.

## Installation

1. Download the latest release of the datapack.
2. Extract the contents into your world save's `datapacks/` directory.
3. Run `/reload` in-game or restart the server to enable the datapack.

## Usage

After installation, all SlimeTools commands are namespaced under
`/function slimetools:`. Use `/function slimetools:help` to list the available
utilities, or consult the `data/slimetools` folder for detailed module
information. Command feedback messages will guide you through each tool's
configuration options.
